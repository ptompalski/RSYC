# Parameters for the RSYC models

A data frame with 1892 rows and 97 columns:

- TileID:

  Tile identified

- SpeciesCode, SpeciesName:

  Species

- b1, b2, b3, b4:

  model parameters

## Usage

``` r
RSYC_params
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with
1892 rows and 7 columns.
